CSS display

HTML 요소가 화면에서 어떤 방식으로 배치되고 표현될지를 결정하는 CSS 속성

block 요소 한 줄 전체 차지 div, p, h1
inline 요소 내용만큼만 차지 span, a, strong

요소의 inline, block 같은 줄을 차지하는 방식, 다른 요소와의 배치관계, 내부 요소의 정렬 방식 등을 결정하는 핵심 속성

1. display: block

            - 한 줄 전체를 차지하며, 항상 줄바꿈 발생
            - 가로폭 100% 차지
            - width와 height 설정 가능
            - 여백 적용 가능

2. display: inline

            - 요소 내용만큼의 크기만 차지하며
            - 줄바꿈 없이 옆으로 배치
            - width와 height 설정 불가능
            - 좌우 여백만 적용 가능
TEXT1 TEXT2 TEXT3

3. display: inline-block

            - block과 inline 특징을 동시에 가짐
            - 줄바꿈없이 옆으로 배치되면서 width와 height 설정 가능
            - 여백 적용 가능

4. display: none

            - 요소를 화면에서 완전히 제거
            - 공간도 차지하지 않음
            - DOM에는 존재
